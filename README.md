## 使用建议
- 因为项目使用eslint husky, 所以建议使用编辑器 vscode 下载插件 ESLint Prettier
- 保存自动格式化，不喜欢的可以在.vscode目录下setting.json修改


## 启动项目
```shell
#首先安装依赖
pnpm install

# 本地运行
pnpm dev

# 打包分析
pnpm analyze

# 打包
pnpm build

```
